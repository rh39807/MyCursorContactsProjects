## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:

npm install

3. Start the development server:

```
npm start
```

The application will be available at `http://localhost:3000`

### Running Tests

```
npm test
```
